# 英语学习推荐材料清单

> 按阶段和类型分类，持续更新。

---

## 第一阶段（第1~2月）

### 精听材料

#### 技术向（难度：中）
| 材料 | 链接 | 说明 |
|------|------|------|
| Microsoft Developer YouTube | youtube.com/@MicrosoftDeveloper | 技术向，语速适中 |
| Fireship YouTube | youtube.com/@Fireship | 短小精悍的技术视频 |
| Traversy Media YouTube | youtube.com/@TraversyMedia | Web 开发教程，讲解清晰 |
| Scott Hanselman YouTube | youtube.com/@shanselman | 技术生态，表达自然 |

#### 通用英语（难度：低-中）
| 材料 | 链接 | 说明 |
|------|------|------|
| TED Talks | ted.com | 有字幕，话题广泛 |
| English with Lucy (YouTube) | youtube.com/@EnglishwithLucy | 英语学习频道 |
| BBC Learning English | bbc.co.uk/learningenglish | 经典学习材料 |

#### 剧集（难度：低）
| 材料 | 说明 |
|------|------|
| Friends | 日常口语，语速适中，梗多 |
| Modern Family | 生活化表达，情景丰富 |
| The Office | 职场英语，幽默表达 |

### 跟读材料
- TED 演讲（选 5~10 分钟的短演讲）
- Friends 精选片段
- YouTube 技术博主的讲解片段

### 阅读材料
| 材料 | 说明 |
|------|------|
| Hacker News (news.ycombinator.com) | 技术社区，短文为主 |
| Reddit r/programming | 技术讨论 |
| Medium 技术文章 | 有难度标注 |
| Dev.to | 开发者博客，友好度高 |
| GitHub README/Issues | 你关注的项目 |

## 第三阶段（第5~6月）

### 高级听力
| 材料 | 说明 |
|------|------|
| Lex Fridman Podcast | 深度 AI/科技访谈 |
| Huberman Lab | 科学类播客 |
| Acquired 播客 | 商业/科技 |
| 技术会议录像（Build、Google I/O） | 技术会议 |
| Google I/O / WWDC | 科技发布会 |

### 高级输出训练
- 用英文写技术博客
- 用英文做 code review
- 用英文写 PR 描述和文档
- 在 Discord/Reddit 参与英文技术讨论
